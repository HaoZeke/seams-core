namespace nneigh
Functions for building neighbour lists. This namespace contains functions that build neighbour lists (using brute-force), saving either the atom IDs or atom indices (according to a PointCloud) in a row-ordered vector of vectors.

#include <src/include/internal/neighbours.hpp>

Parameters
yCloud in The input molSys::PointCloud
nList in Full neighbour list, by atom ID.
Returns Row-ordered full neighbour list, by index, NOT atom ID.

Function for getting the neighbour list by index instead of by atom ID from a previously constructed input neighbour list by ID. The ordering is with respect to the pointCloud with the coordinates.The first element is the atom for which the other atom indices are neighbours For example, if the neighbours of 1 are 2, 3, 4 the sub-vector would have 1 2 3 4
