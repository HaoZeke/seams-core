int clusterAnalysis(std::string path, molSys::PointCloud<molSys::Point<double>, double>* iceCloud, molSys::PointCloud<molSys::Point<double>, double>* yCloud, std::vector<std::vector<int>> nList, std::vector<std::vector<int>>& iceNeighbourList, double cutoff, int firstFrame, std::string bopAnalysis = "q6")
#include <src/include/internal/cluster.hpp>

Parameters
path in Output directory path, specified by the user
iceCloud in/out The molSys::PointCloud for the largest ice cluster of the ice-like molecules
yCloud in The molSys::PointCloud for all the particles in the frame, regardless of ice type
nList in Row-ordered neighbour list by atom ID
iceNeighbourList in Row-ordered neighbour list by atom index, not ID, according to the iceCloud atoms
cutoff in Cutoff for the nearest neighbours
firstFrame in First frame to be analyzed
bopAnalysis in This determines which method to use for determining the ice-like nature of the particles. This can be "q6" or "chill", for using the parameter or CHILL algorithm, respectively

Does the cluster analysis of ice particles in the system. Returns a molSys::PointCloud of the largest ice cluster (using the parameter by default). Uses the full neighbour list (by ID) according to the full PointCloud yCloud. Returns a neighbour list by index, according to the largest ice cluster.
